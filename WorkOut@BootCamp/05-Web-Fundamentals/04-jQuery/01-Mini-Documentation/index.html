Click

Binds an event handler to the "click" JavaScript event, or trigger that event on an element. Example: Clicking the button will display alert box.

Hide

The .hide() method in jQuery is used to conceal elements on a webpage by setting their CSS display property to "none," rendering them invisible. No arguments are required.

Show

The .show() method in jQuery is used to reveal hidden elements on a webpage by changing their CSS display property, making them visible. No arguments are required.

Toggle

The .toggle() method in jQuery is used to alternate between showing and hiding elements on a webpage. It dynamically toggles the visibility of selected elements without requiring any arguments.

Slide Down

The .slideDown() method in jQuery is used to reveal elements on a webpage with a sliding motion. It adjusts the height of the selected elements, gradually displaying them.

Slide Up

The .slideUp() method in jQuery is used to hide elements on a webpage with a sliding motion. It gradually reduces the height of the selected elements, making them disappear.

Slide Toggle

The .slideToggle() method in jQuery is used to toggle the visibility of elements on a webpage with a sliding motion. It alternates between sliding up and down, providing a smooth transition effect.

Fade In

The .fadeIn() method in jQuery is used to gradually reveal hidden elements on a webpage by adjusting their opacity. It provides a smooth fading effect to make elements visible.

Fade Out

The .fadeOut() method in jQuery is used to gradually hide visible elements on a webpage by adjusting their opacity. It provides a smooth fading effect, making elements gradually disappear.

Add Class

The .addClass() method in jQuery is used to add one or more CSS classes to selected elements on a webpage. It modifies the element's class attribute, allowing dynamic styling changes. Example: .bg-blue has been defined in css.

Before

The .before() method in jQuery is used to insert content or elements before the selected elements on a webpage. It allows for dynamic modification of the document structure.

After

The .after() method in jQuery is used to insert content or elements after the selected elements on a webpage. It allows for dynamic modification of the document structure by adding content following the selected elements.

Append

The .append() method in jQuery is used to insert content or elements at the end of the selected elements on a webpage. It adds content inside the specified elements.

The .html() method in jQuery is used to get or set the HTML content of selected elements on a webpage. It allows dynamic manipulation of the element's inner HTML.

Attribute

The .attr() method in jQuery is used to get or set the value of attributes for selected elements on a webpage, allowing dynamic manipulation of element attributes.

Value

The .val() method in jQuery retrieves the current value of form elements, such as input fields. It facilitates dynamic handling of user input within web applications.

Text

The .text() method in jQuery is used to get or set the text content of selected elements on a webpage. It enables dynamic manipulation of element text.
